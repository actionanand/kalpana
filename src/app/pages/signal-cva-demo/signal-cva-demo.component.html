Signal forms

Signal based ControlValueAccessor

This demo uses a reusable custom form control whose internal value, touched state, and disabled state are held in Angular signals while still integrating with Reactive Forms.

What makes it signal based

  • The custom control stores its current value in a signal.
  • writeValue updates that signal when the parent form changes.
  • User selections update the signal and call the registered CVA onChange callback.
  • setDisabledState writes to a disabled signal, so the template reacts immediately.
